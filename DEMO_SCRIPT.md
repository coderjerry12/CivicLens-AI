# CivicLens AI — Demo Script (5 minutes)

## Setup
- Open app at the deployed URL
- Have two browser tabs ready (one for Citizen, one for Authority)
- Prepare a pothole/garbage image on your device

---

## Demo Flow

### 1. Landing Page (30 sec)
- Show the premium landing page
- Point out: AI-powered badge, stats bar, feature cards
- Toggle dark mode
- "This is CivicLens AI — an AI-powered civic platform"

### 2. Citizen Signup (30 sec)
- Click "Get Started"
- Show role selection (Citizen vs Authority)
- Show password strength meter
- "Role-based access from day one"

### 3. Report an Issue (90 sec)
- Click "Report Issue"
- **Voice**: Click mic → speak "there's a large pothole on MG Road"
- Show AI-generated fields
- **OR Photo**: Upload a pothole image
- Show AI analysis animation (detecting type → severity → department)
- Show AI Review panel with editable fields
- Select location on map
- Submit → Show tracking ID + confetti

### 4. Dashboard (30 sec)
- Show personalized greeting
- Live stats from Firestore
- AI insights panel
- Community impact score
- "Everything updates in real-time"

### 5. Authority Workflow (60 sec)
- Switch to Authority tab
- Show Authority Dashboard with dark header
- Open Issue Queue → show priority scores
- Click an issue → show AI analysis + SLA
- Assign to department → notification created
- Mark as "In Progress" → timeline updates
- "Full lifecycle management with AI assistance"

### 6. Community (30 sec)
- Show Community Feed
- Verify an issue (count increments)
- Show notifications arriving in real-time
- "Citizens collaborate to validate issues"

### 7. Intelligence (30 sec)
- Open AI Assistant (chat bubble)
- Ask "summarize my workload"
- Show Analytics with forecasts
- Show GIS Map with hotspots
- "AI-driven decision intelligence"

---

## Key Talking Points

1. **AI-First**: Gemini 2.5 for image analysis, voice reporting, and recommendations
2. **Real-Time**: Firestore listeners, instant notifications, live dashboards
3. **Full Lifecycle**: Report → AI Analyze → Assign → Track → Resolve
4. **Gamification**: Badges, achievements, community impact scoring
5. **Production Quality**: Dark mode, responsive, accessible, TypeScript throughout
